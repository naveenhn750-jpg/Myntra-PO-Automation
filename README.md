# Myntra PO Processor Site

This folder is ready for GitHub Pages or any other static hosting.

## Main files

- `index.html`
- `.nojekyll`
- `.github/workflows/deploy-pages.yml`
- `GITHUB_PAGES_SETUP.md`
- `vercel.json`
- `netlify.toml`

## Recommended publish option

GitHub Pages is a good fit when you want:

- a free public URL
- simple static hosting
- an easy repo-based workflow for updates

## Expected behavior

- Upload one master workbook
- Upload all PO workbooks
- Generate the final mapped Excel output in the browser

## Note

The page loads the spreadsheet library from a CDN at runtime, so the deployed site needs normal internet access.
